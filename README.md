## About The Project


Aplikasi eAPD merupakan aplikasi berbasis web untuk pendataan APD seluruh pegawai Dinas Penanggulangan Kebakaran dan Penyelamatan Provinsi DKI Jakarta.



### Built With

Framework, dan Library yang digunakan :

* Code Igniter 3
* TC-PDF
* Bootstrap
* Jquery
* Data Table
* Chart JS
* Compressor JS
* Mobilekit
* Admin Circle
